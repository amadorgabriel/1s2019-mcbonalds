<p align="center"> Sistema completo que gerencia pedidos na hamburgueria </p>


## 📝 Projeto:
- Login e Logout
- Cadastar usuários;
- Listar Shakes;

- Solicitar pedido;

## 🧠 Conhecimentos trabalhados
- ASP.NET MVC 5 
- DotNet
- Razor Syntax
- Banco de dados em .CSV
